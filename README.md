# Typing Speed Test App

A modern **Typing Speed Test Web App** built using **JavaScript ES6, HTML, and CSS**.
Test your typing speed, accuracy, and performance in a clean and responsive interface.

---

##  Live Demo

👉 https://sutirthaghosh04.github.io/typing-speed-test/

---


##  Features

*  60-second countdown timer
*  Words Per Minute (WPM) calculation
*  Accuracy percentage
*  Total words typed
*  Multiple random paragraphs
*  Dark mode toggle
*  Restart test option
*  Auto stop when time ends or text completes
*  Full-screen responsive UI


```
cd typing-speed-test
```

### 3. Run the app

Open `index.html` in your browser

---

##  How It Works

1. Click **Start Test**
2. A random paragraph appears
3. Start typing in the input box
4. The app calculates:

   * Speed (WPM)
   * Accuracy (%)
   * Words typed
5. Test ends when:

   * Time runs out ⏳
   * OR paragraph is completed
